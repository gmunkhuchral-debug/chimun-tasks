## PIN-н бүтэц

PIN-ийн эхний орон нь бүлгийг илэрхийлнэ:
- `1xxx` — Нэгдсэн алба (CEO 1234 онцгой)
- `2xxx` — M Event салбар
- `3xxx` — NOMAAD Camp
- `4xxx` — NOMAAD Store

Шалтгаан: бүлэг сольсон ажилтны PIN-г шинэчлэхэд хялбар. Шинэ ажилтан орвол тухайн бүлгийн дараагийн дугаарыг өгнө.

## Ажилтнуудад илгээх template

Ажилтан бүрд WhatsApp/Telegram-аар илгээх бичвэр (нэр + PIN-г солих):

```
Сайн уу [НЭР],

Чимун ХХК-ийн дотоод даалгавар апп бэлэн боллоо. Утсаа авч 
дараах линкийг Safari-ээр нээ:

https://gmunkhuchral-debug.github.io/-/

Доод цэснийн Share товч (□↑) → "Add to Home Screen" дарна. 
Үндсэн дэлгэцэд апп шиг суусан байна.

Нэвтрэх:
- Хэн бэ? → өөрийгөө сонго
- PIN: [PIN]

PIN-г нууц байлгана уу. Бусдад бүү хэл.
```

## Нэвтрэх алхам (ажилтанд тайлбарлах)

1. Утсан дээрх "Чимун Tasks" аппын иконыг даран нээ
2. "Хэн бэ?" дроп → өөрийгөө сонгох
3. PIN кодоо оруулах (4-оронтой автомат submit хийгдэнэ)
4. Бүх даалгаврууд харагдана. Зөвхөн өөрт нь оноогдсонг харна

24 цагт нэг л удаа нэвтрэхэд хангалттай — апп тэр хэрэглэгчийг санана. 24 цаг өнгөрсний дараа дахин PIN оруулна.

## PIN солих

PIN-г өөрчлөх хэрэгтэй болвол `index.html`-н `TEAM` constant дотор тухайн хүний `pin:` талбарыг солих → GitHub-руу push хийнэ. Жишээ: `pin: '2001'` → `pin: '7843'`. 5 минутад апп шинэчлэгдэнэ.

> **Аюулгүй байдлын тэмдэглэл:** Энэ нь production-grade auth биш — PIN кодууд `index.html` файлын JavaScript хэсэгт ил харагдана (browser-н view-source-аар). Дотоод компанийн жижиг систем учир ажилтнуудын хүн ам цөөн, итгэмжлэлтэй харьцаатай тул хангалттай. Хэрэв илүү аюулгүй болгох хэрэгтэй бол n8n + Google Sheets backend-аар нууцлал хадгалах хувилбарт шилжих боломжтой.
