2. C01 Дэлгэрбат-ын **Хавсарсан бүлэг** cell (E12) дотроос `Store` гэсэн утгыг устгаад **хоосон** болгох

## 1. Header мөрийг "freeze" хийх (5 секунд)

1. Дээд цэснээс `View` → `Freeze` → `1 row`

Эхний мөр доош гүйлгээж байсан ч үргэлж дээр харагдана.

---

## 2. "Бүлэг" багана (D) — dropdown суулгах

1. D баганын title cell (`Бүлэг`) дээр товшоод бүх баганыг сонгох
2. `Data` → `Data validation` → `+ Add rule`
3. **Apply to range:** `D2:D` (header-ээс доош бүх cell)
4. **Criteria:** `Dropdown` сонгох
5. Add options:
   - `Нэгдсэн`
   - `M Event`
   - `Camp`
6. `Show a warning` сонгох (буруу утга оруулсан үед warning)
7. `Done`

---

## 6. "Үндсэн цалин" багана (N) — Currency формат

1. N баганыг бүхэлд нь сонгох
2. `Format` → `Number` → `Currency`
3. Доош нь scroll хийгээд `Custom currency` сонгох
4. Хэрэв `MNT` олдохгүй бол: `Custom number format` → `[$₮]#,##0;[red][$₮]-#,##0` бичих
5. `Apply`

Одоо `1800000` гэж бичсэн нь `₮1,800,000` гэж харагдана.

---

## 7. "Огноо" багана-уудад (K, L, M) — Date формат

K (Ажилд орсон), L (Улирлын эхлэл), M (Улирлын төгсгөл) баганууд бүгдийг сонгох → `Format` → `Number` → `Date` (`YYYY-MM-DD` format).

---

## 8. Conditional formatting — "гарсан" болон "нээлттэй" мөрийг харагдуулах

1. Бүх мөрийг сонгох (A2:R)
2. `Format` → `Conditional formatting`
3. **Rule 1:** Custom formula `=$J2="гарсан"` → background color: **саарал**, font: **strikethrough**
4. **Rule 2:** Custom formula `=$J2="нээлттэй"` → background color: **шар (бараан)**, font: **italic**
5. **Rule 3:** Custom formula `=$J2="улирлын"` → background color: **цэнхэр-чийр (бараан)**
6. `Done`

Үүний дараа Sheet нь шууд харахад нэг үзэгдэлд: "хэн идэвхтэй, хэн улирлын, хэн нээлттэй" гэдгийг ялгаж харна.
