# 📱 Утсанд суулгах заавар

Энэ апп **PWA (Progressive Web App)** учраас App Store / Google Play-аас татах шаардлагагүй. Ажилтан утсаараа URL-ийг нэг л удаа нээгээд "Home Screen-д нэм" дарна. Тэр цагаас хойш утасны үндсэн дэлгэц дээр иконтой, native апп шиг ажиллана.

```
┌──────────────────────────────────────────────────┐
│ 1. Чи (CEO) — GitHub Pages дээр upload (10 мин)  │
│         ↓                                        │
│ 2. Багт URL илгээх (1 мин)                       │
│         ↓                                        │
│ 3. Ажилтан URL нээж "Home Screen-д нэм" дарна   │
│         ↓                                        │
│ 4. Утсанд апп шиг иконтой суусан байна 🎉        │
└──────────────────────────────────────────────────┘
```

## ① 10 минутад publish хийх — GitHub Pages

### А. GitHub account үүсгэх (хэрэв байхгүй бол)

1. https://github.com/signup → имэйл + нууц үг
2. Username сонгох (жишээ: `chimun-hxk`, `m-event-mongolia`)
3. Имэйл баталгаажуулна

### Б. Шинэ repository үүсгэх

1. https://github.com/new
2. **Repository name:** `chimun-tasks` (эсвэл өөр сонголт)
3. **Public** сонгоно (GitHub Pages үнэгүй ажиллахын тулд)
4. **Add a README file** чагтлахгүй
5. **Create repository** дарна

### В. Файлуудыг upload хийх

1. Шинэ repo хуудсан дээрх **"uploading an existing file"** холбоосыг дарна (эсвэл `Add file → Upload files`)
2. Finder нээгээд `Desktop/web site & app/АПП ЧИМУН ХХК/` фолдероос дараах **9 файлыг** drag-and-drop хийнэ:

   ```
   ✅ index.html
   ✅ manifest.json
   ✅ sw.js
   ✅ icon.svg
   ✅ icon-192.png
   ✅ icon-512.png
   ✅ icon-maskable-512.png
   ✅ apple-touch-icon.png
   ✅ README.md
   ```

   > **Анхаар:** `АРХИТЕКТУР-ТӨЛӨВЛӨГӨӨ.md`, `УТСАНД-СУУЛГАХ.md`, `n8n-workflow-*.json`, screenshot PNG-нүүд upload хийх шаардлагагүй (эдгээр нь дотоод бичиг баримт, апп ажиллахад хэрэггүй).

3. Доод талд **Commit changes** дарна

### Г. GitHub Pages идэвхжүүлэх

1. Repo дотор → **Settings** таб
2. Зүүн дээр → **Pages**
3. **Source** → "Deploy from a branch"
4. **Branch** → `main` сонгоод `/ (root)` → **Save**
5. 1-2 минут хүлээгээд хуудсыг refresh хийгээрэй
6. Дээр **"Your site is live at https://USERNAME.github.io/chimun-tasks/"** гэж гарна

**Энэ URL-ийг хуулж аваарай** — багт энэ URL-ийг л өгнө.

## ③ Утсанд "Home Screen-д нэм" хийх заавар

### iPhone / iPad (Safari)

1. **Safari-д** URL нээх (Chrome дээр ажиллахгүй — заавал Safari)
2. Доод цэснээс **Share товч** (□↑) → дарна
3. Доош scroll хийгээд **"Add to Home Screen"** (Үндсэн дэлгэцэд нэмэх)
4. Apps' name-г "Чимун Tasks" гэж үлдээх эсвэл өөрчилөх
5. **Add** дарна
6. ✅ Үндсэн дэлгэц дээр иконтой апп шиг гарсан байна

### Android (Chrome)

**Хувилбар A — Самбар автоматаар санал болгоно:**
1. Chrome-д URL нээгээд хэдхэн секунд хүлээнэ
2. Доод хэсэгт **"Install Чимун Tasks"** banner гарвал дарна
3. **Install** дарна

**Хувилбар B — Гараар:**
1. Chrome-н дээр баруун буланд **3-цэгийн товч (⋮)** дарна
2. **"Install app"** эсвэл **"Add to Home screen"** сонгоно
3. **Install** дарна
4. ✅ Үндсэн дэлгэцэд иконтой апп гарсан байна

> **Тэмдэглэл:** Android дээр Firefox болон бусад browser-д ч ажиллана, гэхдээ Chrome-д хамгийн зөв ажиллана.

## ④ Хэрхэн ажиллах вэ?

### Татсаны дараа (анх 1 удаа)

| Алхам | Юу хийх |
|---|---|
| 1 | "Бүлэг" dropdown-аас өөрийн ажилладаг бүлгийг сонго |
| 2 | "Би хэн бэ?" хэсэгт өөрийгөө сонгох |
| 3 | "+ Шинэ" товчоор анхны даалгавраа нэмж үзнэ |

### Өдөр тутам

- **🏠 Home screen-н иконоо дарна** — апп нь browser шиг бус шууд нээгдэнэ
- **+ Шинэ** товчоор даалгавар нэмнэ
- **✓** товчоор биелсэн гэж тэмдэглэнэ
- **Миний** хэсгээс өөрийн даалгавруудыг харна

### Offline ажиллах

Service worker нэмсэн учраас интернэтгүй үед ч апп нээгдэнэ. Хадгалсан data нь дараа интернэт ороход syncиглэгдэнэ (одоохондоо backend сэтгэх ажил үргэлжилсэн).

---

## ⑤ Шинэ хувилбар гарах үед

Чи `index.html`-г засаад GitHub repo-д commit хийхэд **5-10 минутад** бүх ажилтны утсан дээрх апп автоматаар шинэчлэгдэнэ. Service worker `CACHE_VERSION` шалгаад өөрчлөгдсөн файлуудыг шинэчилнэ.

> Хэрэв ажилтнуудад шинэчлэлт хүрэхгүй бол `sw.js` доторх `CACHE_VERSION` тоог нэмж commit хий — энэ нь cache busting-н хувилбарыг ахиулна.

## ⑥ n8n + Google Sheets sync (заавал биш, дараа)

Одоохондоо local mode (зөвхөн өөрийн утсанд хадгалагдана). Олон ажилтан нэг ижил data харахын тулд n8n + Google Sheets backend-ыг setup хийнэ. Жагсаал заавар нь **README.md** файлд бий, дараах алхам:

1. Google Sheet `M Event Tasks` нэрээр үүсгэх (`tasks` хуудастай)
2. n8n дээр `n8n-workflow-sync.json` болон `n8n-workflow-notify.json` import хийх
3. Үүсгэсэн webhook URL-ийг апп → "⚙ Тохиргоо" хэсэгт оруулах

> **Анхаар:** Одоогийн n8n workflow-уудад 4 алдаа байгаа (өмнөх review-д тогтоосон). Production-д тавихаас өмнө засах ёстой.

## ⑦ Асуудал гарвал

| Асуудал | Шийдэл |
|---|---|
| URL нээгдэхгүй | URL зөв уу шалга. `https://USERNAME.github.io/chimun-tasks/` хэлбэртэй байх ёстой |

| Локалд (file://) PWA ажиллахгүй | PWA нь зөвхөн `https://` дээр ажилладаг. GitHub Pages дээр upload хийсэн URL ашиглах ёстой |
